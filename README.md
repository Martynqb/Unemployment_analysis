# Unemployment Analysis with Python 📊

## 📌 Project Overview
This project analyzes unemployment trends across different regions using real dataset.

## 🎯 Objective
To understand how unemployment rates change over time and compare between regions.

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib

## 📂 Dataset
Custom dataset with unemployment rates for India and USA (2018–2022)

## 📊 Analysis Performed
- Data loading and cleaning
- Grouping by region
- Calculating average unemployment rate
- Visualization using line graphs

## 📈 Insights
- Unemployment peaked during 2020 (COVID period)
- USA shows lower unemployment compared to India
- Trend decreases after 2020
